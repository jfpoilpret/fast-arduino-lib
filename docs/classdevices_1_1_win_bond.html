◆ wait_until_ready()

template<board::DigitalPin CS>
bool devices::WinBond< CS >::wait_until_ready ( uint16_t  timeout_ms)

Wait until any erase or write operation is finished.

This method continuously reads the Status register and check the BUSY bit (§6.1.1). If the chip is busy, then the method yields time, i.e. put the MCU to sleep according to the default board::SleepMode.

Parameters
timeout_msthe maximum time, in milliseconds, to wait for the chip to be ready
Return values
trueif the chip is ready
falseif the chip is still busy after timeout_ms delay
See also
time::yield()
power::Power::set_default_mode()
