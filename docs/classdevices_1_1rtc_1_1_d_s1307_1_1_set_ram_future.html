Detailed Description

template<typename MANAGER>
template<uint8_t SIZE_>
class devices::rtc::DS1307< MANAGER >::SetRamFuture< SIZE_ >

Create a future to be used by asynchronous method set_ram(SetRamFuture<SIZE>&).

This is used by set_ram() to set device data, and it shall be used by the caller to determine when the I2C transaction is finished.

Template Parameters
SIZE_the number of bytes to set to the chip RAM
Parameters
addressthe address of the first cell to set, must be between 0 and ram_size().
dataa point to a buffer of size SIZE_ which content will be written to the chip RAM starting at address
