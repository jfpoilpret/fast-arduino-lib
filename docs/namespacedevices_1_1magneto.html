◆ DLPF

enum devices::magneto::DLPF : uint8_t
strong

The Digital Low Pass Filter bandwidth to select for the chip (register map §4.3).

This can be expressed either from the gyroscope viewpoint ot from the accelerometer viewpoint, but any setting is common to both features, i.e. selecting a DLPF setting for the accelerometer will force the matching setting for the gyroscope.
