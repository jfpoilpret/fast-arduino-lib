Detailed Description

template<typename MANAGER>
class devices::rtc::DS1307< MANAGER >::HaltClockFuture

Create a future to be used by asynchronous method halt_clock(HaltClockFuture&).

This is used by halt_clock() to set device data, and it shall be used by the caller to determine when the I2C transaction is finished.
