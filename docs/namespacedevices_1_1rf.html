Detailed Description

Defines the API for radio-frequency (wireless) communication support.

Current support is for Nordic Semiconductor nRF24L01+ chip (SPI-based). API is provided in 2 flavours, IRQ-based or not, through two template classes. IRQ-based flavour is encouraged but is sometimes impossible as it requires an available board::ExternalInterruptPin on the MCU.
